## Описание проекта

Простой Telegram‑бот для изучения слов иностранного языка. Бот не требует отдельного сайта: всё общение происходит прямо в Telegram.  
Пользователь может добавлять свои слова или брать готовый набор, получать напоминания о повторении, проходить тренировки по карточкам и смотреть статистику прогресса.

## Архитектура и технология

- **Технологии**
  - Python (монолитное приложение)
  - Telegram‑бот на `aiogram`
  - База данных: PostgreSQL или SQLite
  - Фоновые задачи для напоминаний о повторении слов

- **Компоненты приложения**
  - **Telegram‑бот**: обработка команд, сообщений и сценариев общения.
  - **Модуль работы с БД**: взаимодействие с PostgreSQL/SQLite.
  - **Модуль бизнес‑логики**: словарь, тренировки, подсчёт прогресса.
  - **Модуль фоновой задачи**: планировщик и рассылка напоминаний пользователям.

- **Инфраструктура**
  - Приложение и база данных поднимаются через `docker-compose`.
  - Один контейнер с приложением и один контейнер с БД.

## Поведение для пользователя

### Первый запуск

- **Приветствие**: бот здоровается и сохраняет `user_id`.
- **Настройки**:
  - выбор языка обучения;
  - указание примерного уровня (A1–C1);
  - выбор частоты напоминаний о повторении слов.

### Добавление слов

- **Готовый набор**: пользователь может взять небольшой стартовый набор базовых слов.
- **Свои слова**: можно добавлять слова в формате `слово — перевод`.

### Тренировка

- **Запуск тренировки**: по команде «Начать тренировку» бот показывает карточки.
- **Ответы пользователя**:
  - ответы вариантами (кнопки);
  - ответы текстом.
- **Обратная связь**:
  - бот показывает правильный ответ;
  - фиксирует результат для дальнейшего расчёта прогресса и повторений.

### Напоминания

- **Фоновый процесс**:
  - раз в заданный пользователем интервал находит слова, которые пора повторить;
  - отправляет пользователю напоминание в Telegram.
- **Короткая сессия**: бот предлагает пройти быструю тренировку по словам, которые «созрели» для повторения.

### Прогресс

- **Что хранится**:
  - количество выученных слов;
  - число проведённых тренировок;
  - серия активных дней (streak).
- **Команда «Мой прогресс»**:
  - выводит основную статистику по изучению слов и активности пользователя.

## Основной функционал (MVP)

- **Регистрация пользователя и его настроек**
  - Сохранение `user_id`, языка обучения, уровня и частоты напоминаний.

- **Хранение слов и простого прогресса по каждому слову**
  - Словарные записи с переводом.
  - Статус и история ответов для базового алгоритма повторения.

- **Режим тренировки с карточками**
  - Показ слов/переводов в виде карточек.
  - Сбор ответов и фиксация результатов.

- **Напоминания о повторении через фоновые задачи**
  - Планировщик, который регулярно проверяет, какие слова пора повторить.
  - Отправка уведомлений и запуск мини‑тренировок.

- **Просмотр базовой статистики**
  - Команда для просмотра количества выученных слов, числа тренировок и серии активных дней.

## Распределение ролей в команде

- **Аркадий — NLP / ML**
  - Продумывает и реализует логику повторения слов: как часто и в какой последовательности показывать слово в зависимости от ответов пользователя.
  - Готовит идеи по более продвинутым алгоритмам интервального повторения и возможному дальнейшему усложнению системы.

- **Ксения — Бизнес‑логика и база данных**
  - Проектирует структуру БД: пользователи, слова, настройки, прогресс.
  - Пишет сервисы для добавления слов, сохранения результатов тренировок и подсчёта статистики.

- **Андрей — Сценарии и тексты внутри бота**
  - Придумывает сценарии общения: приветствие, меню, тексты команд и подсказок.
  - Описывает форматы тренировок: вид вопросов, варианты ответов и сообщения после них.

- **Егор — Telegram‑бот на aiogram**
  - Реализует бот: команды, кнопки, переходы между состояниями.
  - Связывает интерфейс Telegram с бизнес‑логикой Ксении и алгоритмами Аркадия.
  - Проверяет, что основные пользовательские сценарии работают стабильно.

- **Даня — Инфраструктура, фон и документация**
  - Настраивает проект, Docker и `docker-compose`.
  - Реализует фоновый планировщик, который рассылает напоминания.
  - Пишет инструкции по запуску сервиса и делает простую архитектурную схему для отчёта.

## Возможные направления развития

Эти фичи не входят в MVP, но могут быть реализованы как последующие шаги:

- **Обучение через контент**
  - Привязка слов к фильмам, сериалам, книгам и музыке.
  - Тренировки по словам из конкретных произведений.

- **Продвинутый алгоритм интервального повторения**
  - Более точные модели оценки забывания.
  - Настройка расписаний повторения под стиль обучения пользователя.

- **Диалоги‑персонажи и языковые модели**
  - Диалоговые сценарии с «персонажами» для практики.
  - Разбор письменных ответов с помощью языковых моделей (LLM).

- **Новые режимы обучения**
  - Отдельные режимы грамматики, аудирования и говорения.
  - Режим подготовки к экзаменам (IELTS, Goethe и др.).

## Кратко

Бот закрывает базовые задачи изучения слов: добавление и хранение слов, тренировки с карточками, напоминания о повторении и просмотр прогресса.  
Архитектура проста (монолит на Python с БД и фоновой задачей), а команда распределена по ролям так, чтобы быстро довести MVP до рабочего состояния и иметь понятный путь развития проекта.
